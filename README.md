# Linux VoIP Lab

A personal VoIP lab built using Debian Linux and KVM virtualization.

## Environment

* Host OS: Debian
* Virtualization: KVM/QEMU
* VM Name: voipserver
* Guest OS: Debian 13

## Objectives

* Learn VoIP fundamentals
* Install Asterisk
* Configure SIP extensions
* Configure IVR
* Learn call routing
* Maintain documentation in GitHub

## Lab Nodes

* node1 = webserver
* node2 = dataserver
* node3 = voipserver
